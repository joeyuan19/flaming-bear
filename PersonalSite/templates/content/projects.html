PHYSICS SIMULATIONS

Masses on a String
A simulation of a string with tension and masses placed at equal separation along the string. Contains controls to adjust the physical properties and environment varaibles. Keyboard controls are enabled. Requires plugins to support running Java applets. Alternatively you can download the source files here. Written in Java.

PUZZLES

Sudoku Solver
Uses logical elimination techniques to solve sudoku puzzles. This implemenation is written in Java in order to display it on the web. There are also versions I have written in Python as a command line application and also as a GUI '.app' version.
